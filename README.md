# Training and Certification Tracking System

A comprehensive platform to track employee training programs, certifications, and validity periods.

## Features

-   Training enrollment and completion tracking
-   Certification expiry monitoring
-   Compliance reporting APIs
-   Real-time notifications for expiring certifications

## Tech Stack

-   **Backend**: FastAPI, Python, SQLAlchemy, MySQL
-   **Frontend**: React, TypeScript, Tailwind CSS
-   **Database**: MySQL
-   **Testing**: pytest
